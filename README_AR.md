# OMEGA PRIME Fusion MVP v0.1

## التشغيل الفوري على الهاتف أو الكمبيوتر
1. فك الضغط.
2. افتح `index.html` في المتصفح.
3. اضغط **تشغيل مجلس المديرين**.
4. عدّل الكميات والمتوسطات والأسعار داخل **المحفظة التنفيذية**؛ يعاد الحساب مباشرة.
5. استخدم **تصدير الجلسة** لحفظ الأوامر والسجل وبيانات المحفظة في JSON.

## ما يعمل الآن
- واجهة عربية RTL حقيقية.
- ثمانية مديرين ظاهرين ويعملون ضمن تسلسل مجلس.
- محرك قواعد محلي لا يحتاج API.
- حساب الربح/الخسارة والتركيز والمخاطر.
- مناطق بناء، وقف مرجعي، أهداف، وقرار لكل أصل.
- أوامر تنفيذية وخطة تدوير للسيولة.
- تصدير الجلسة.

## الترقية إلى AI حقيقي
المجلد `backend` يحتوي هيكلًا أوليًا باستخدام OpenAI Agents SDK.
بعد إضافة مفتاح API يمكن استبدال محرك القواعد المحلي بوكلاء AI مع handoffs وtracing.

## تنبيه
النسخة الحالية أداة دعم قرار واختبار هندسي. الأسعار ليست حية، والمستويات المحسوبة آلية مرجعية وليست توصية شخصية أو أمر تداول.
